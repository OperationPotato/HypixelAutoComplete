# Hypixel Tab Completions

A simple 1.21.5 Fabric mod to add tab completions for various Hypixel commands.

## Supported Commands

|  Name   |   Aliases    |
|:-------:|:------------:|
|  /chat  |              |
| /friend | /f, /friends |
|   /fl   |              |
| /guild  |      /g      |
| /party  |      /p      |
| /status |              |
